# **2ND SEMESTER**
For this second semester, I learn these subjects :
## **1. Data Structure and Object-Oriented Programming**
The Data Structures and Object-Oriented Programming course covers concepts and 
efficient techniques for organizing, storing, accessing, and manipulating data within a 
computer program. Additionally, this course provides an understanding of the fundamental principles of 
object-oriented programming OOP paradigm, including the use of classes, objects, 
attributes, and methods, as well as core OOP concepts such as abstraction, 
encapsulation, inheritance, and polymorphism.
## **2. Operating Systems**
After learning the basic structure of Computer Architecture in 1st Semester, The Next Stop is Operating Systems.
It consists of ways to understand the Operating Systems through a series of handling the computer's resource up to Shell-Scripting using Linux OS (Lab Session).
## **3. Database Systems**
After learning about Statistics and Probabilities, We Shift into learning how to manage Database Systems.
It is a very core concept to learn how to be a Backend Dev.
## **4. Enterprise Architecture**
The Departement of IT is generous enough to give the IT Students ways to bridge the gap in IT and Business.
This Course burdens us heavily on the TOGAF Architecture concept, the main way to bridge the previously mentioned gap.
## **5. Human and Computer Interaction**
This Course is supposed to be taken at fourth semester, but i am delighted to be one of the few students to take it early.
It gives insights on how to design not only a good-looking app/web, but also surrounding how one experiences it (UI/UX).
## **6. Calculus II**
Not related to programming at all, this course is a continuation of Calculus I and is heavily focused around the Derivative and Integration.
